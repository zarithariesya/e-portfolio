Career Exploration

Chosen Career

System Administrator

A System Administrator is responsible for maintaining, managing and securing computer systems, servers and network infrastructures within an organization. This role is important in modern cybersecurity because System Administrators help protect systems from cyber threats, ensure data availability and maintain smooth IT operations.

Job Responsibilities

  • Installing and configuring operating systems
  • Managing servers and networks
  • Monitoring system performance
  • Performing backup and recovery
  • Managing user accounts and permissions
  • Updating software and security patches
  • Troubleshooting hardware and software issues
  • Ensuring cybersecurity measures are implemented
